# Arithmetic Evaluator
## Desription
Simple **arithmetic calculator** with support for some math functions.
Basic operations: +, -, *, /, %, ^;

Supports following functions:
- max
- min
- pow
- sqrt
- sin
- cos
- tan
- signum
- abs
- exp
- log
- log10
- floor
- ceil

## Examples
```
pow(max(5, 2), min(2, 5))
-1 - 5 - 8 -73+ log10(10000)*signum(-10)
123 % 0
```
